# Spec Index（スペック一覧）

全スペックの一覧と状態を管理するインデックス。
スペック追加・更新時は必ずこのファイルも更新する。

## 一覧

| ID | ファイル | タイトル | ステータス | 依存 | 最終更新 |
|----|----------|----------|------------|------|----------|
| - | - | （スペック未作成） | - | - | - |

## ステータス定義

| ステータス | 説明 |
|------------|------|
| draft | 下書き。レビュー未実施 |
| review | レビュー中 |
| approved | 承認済み。実装可能 |
| implemented | 実装完了。テスト通過 |
| deprecated | 廃止 |

## 依存関係ルール

- 依存先のスペックが `approved` 以上であること
- 循環依存は禁止
- 依存先が `deprecated` になった場合は影響を確認

## 更新手順

1. 新規スペック作成時: 行を追加し `draft` に設定
2. レビュー開始時: `review` に更新
3. 承認時: `approved` に更新
4. 実装完了時: `implemented` に更新
5. 廃止時: `deprecated` に更新（理由を備考に記載）
